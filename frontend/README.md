# TestShopp Frontend

Vue 3 + Vite + Tailwind CSS — Persian RTL static demo.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
# Local preview
npm run build

# GitHub Pages (base: /testshopp/)
npm run build:pages
```

## Pages (Phase 1)

| Route | Page |
|-------|------|
| `/` | Home — hero, categories, products |
| `/auth/login` | Login |
| `/auth/register` | Register |
| `/auth/forgot-password` | Forgot password |

## Features

- Persian RTL layout with Vazirmatn font
- Dark mode toggle
- Responsive mobile-first design
- Toast notifications
- Cart & wishlist (client-side demo)
- Static product data
